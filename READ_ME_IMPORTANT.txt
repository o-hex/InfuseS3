Hello! Thank you for downloading my Infuse project.

You might be wondering why there are two folders with seemingly identical content. This is to ensure compatibility with different versions of Skript and SkBee.

- infuse_old: For Skript 2.14.2 and SkBee 3.16.1 (1.21.4 - 1.21.11)
- infuse_new: For Skript 2.15.0 and SkBee 3.17.0+ (1.21.4+, such as 1.21.6, 1.21.8, 1.21.11, or 26.1)

If you are using 1.21.4, or are restricted to Skript 2.14.2, please use "infuse_old".
If you are using 1.21.4+ (such as 1.21.6, 1.21.8, 1.21.11, or 26.1), please use "infuse_new".

"infuse_new" introduces MiniMessage support, making it easier to read and debug. 
There is no significant performance difference between the two, so you can decide for yourself.

The resource pack has been tested on versions 1.21.4, 1.21.8, and 1.21.11.
This project has been extensively tested on 1.21.11 and is specifically designed for that version.

Installation Method:
Place the entire folder into /plugins/Skript/scripts.
The path should be /plugins/Skript/scripts/infuse_xxx

Note: Do not put the files individually into the scripts folder; some scripts have identical names which will cause conflicts.
